Formatação para códigos

Utilizando a tag <code> declaramos que o texto representa um código e deixamos as letras mono-espaçadas, com o uso do <pre>, deixamos a formatação do jeito que foi escrito no código html. Exemplo de código em python: 

    x = 5
    if x < 9:
        print("Olá!")
    elif x < 15:
        print("Que bom ver você!")
    else:
        print("Tchau!")
            
    print("Fim")
